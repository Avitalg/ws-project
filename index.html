    Welcome to GitHub Pages.

    This is our formal API for Node.js service that was bulid with express module for a school project.

    Looks

    Get All Looks

    URL: https://webserviceproj.herokuapp.com/api/getAllLooks Result: All looks, including their steps Method: GET Example: 

    {
  look: "fresh",
  image: "http://res.cloudinary.com/desbjknxm/image/upload/v1468704051/rsz_3fresh_eovzbd.jpg",
  __v: 13,
  steps: [
  {
    number: 1,
    face_image: "http://res.cloudinary.com/desbjknxm/image/upload/v1468700970/htijb01e02otjh6ozf24.jpg",
    description: "description",
    product_id: 4
  }]
}

    Get Look

    URL: https://webserviceproj.herokuapp.com/api/getLook/:look Result: Specific Look. Method: GET Example: Like above.

    Add Look

    URL: https://webserviceproj.herokuapp.com/api/addLook Result: JSON with message - success/error. Method: POST Params: look, url.

    Add Look

    URL: https://webserviceproj.herokuapp.com/api/updateLook Result: JSON with message - success/error. Method: POST Params: look, url.

    Remove Look

    URL: https://webserviceproj.herokuapp.com/api/removeLook/:look Result: JSON with message - success/error. Method: GET Params: look.

    Add Step To Look

    URL: https://webserviceproj.herokuapp.com/api/addStepToLook Result: JSON with message - success/error. Method: POST Params: look, number, url, desc, prodId. Explain: look- look's name, number - step number, url -image, desc - description, prodId- product id.

    Remove Step From Look

    URL: https://webserviceproj.herokuapp.com/api/removeStepFromLook/:look/:number Result: JSON with message - success/error. Method: GET Params: look-name of look, number- step number.

    Remove Step From Look

    URL: https://webserviceproj.herokuapp.com/api/getLookSteps/:look Result: JSON with message - success/error. Method: GET

    CATEGORIES

    Get All Categories

    URL: https://webserviceproj.herokuapp.com/api/getAllCategories Result: JSON all categories that was saved in DB. Method: GET Example:

    [{
  _id: "578938251d4e340300fd3e81",
  id: 1,
  name: "eyes",
  image: "https:/res.cloudinary.com/desbjknxm/image/upload/v1468610599/piecsqzxmknbnzbinxed.jpg",
  __v: 0
}]

    Get Category

    URL: https://webserviceproj.herokuapp.com/api/getCategory/:category Result: JSON specific category, like shown above. Method: GET

    Remove Category

    URL: https://webserviceproj.herokuapp.com/api/removeCategory/:category Result: JSON with message - success/error. Method: GET.

    Add Category

    URL: https://webserviceproj.herokuapp.com/api/addCategory/:id/:name Result: JSON with message - success/error. Method: GET.

    Update Category

    URL: https://webserviceproj.herokuapp.com/api/updateCategory/:id/:name Result: JSON with message - success/error. Method: GET.

    Update Category

    URL: https://webserviceproj.herokuapp.com/api/uploadCategoryImage Result: JSON with message - success/error. Method: POST. Params: category - category name, url - url to image.

    USER

    Get All Users

    URL: https://webserviceproj.herokuapp.com/api/getUsers Result: JSON with all users that was saved in DB. Method: GET Example:

    [
  _id: "578a0ad1fa8650030052eedb",
  username: "tryt5475@gmail.com",
  admin: false,
  __v: 0,
  wish_list: [ ]
]

    Get User

    URL: https://webserviceproj.herokuapp.com/api/getUser/:username Result: JSON specific user, like shown above. Method: GET

    Remove User

    URL: https://webserviceproj.herokuapp.com/api/removeUser/:username Result: JSON with message - success/error. Method: GET.

    Add User

    URL: https://webserviceproj.herokuapp.com/api/addUser/:username/:admin Result: JSON with message - success/error. Method: GET. Params: username- email, admin- boolean.

    Update User

    URL: https://webserviceproj.herokuapp.com/api/updateUser/:username/:admin Result: JSON with message - success/error. Method: GET.
