Welcome to GitHub Pages.

This is our formal API for Node.js service that was bulid with express module for a school project.

Looks

Get All Looks

URL: https://webserviceproj.herokuapp.com/api/getAllLooks
Result: All looks, including their steps
Method: GET
Example: 

{
  look: "fresh",
  image: "http://res.cloudinary.com/desbjknxm/image/upload/v1468704051/rsz_3fresh_eovzbd.jpg",
  __v: 13,
  steps: [
  {
    number: 1,
    face_image: "http://res.cloudinary.com/desbjknxm/image/upload/v1468700970/htijb01e02otjh6ozf24.jpg",
    description: "description",
    product_id: 4
  }]
}

Get Look

URL: https://webserviceproj.herokuapp.com/api/getLook/:look
Result: Specific Look.
Method: GET
Example: Like above.

Add Look

URL: https://webserviceproj.herokuapp.com/api/addLook
Result: JSON with message - success/error.
Method: POST
Params: look, url.

Add Look

URL: https://webserviceproj.herokuapp.com/api/updateLook
Result: JSON with message - success/error.
Method: POST
Params: look, url.

Remove Look

URL: https://webserviceproj.herokuapp.com/api/removeLook/:look
Result: JSON with message - success/error.
Method: GET
Params: look.

Add Step To Look

URL: https://webserviceproj.herokuapp.com/api/addStepToLook
Result: JSON with message - success/error.
Method: POST
Params: look, number, url, desc, prodId.
Explain: look- look's name, number - step number, url -image, desc - description, prodId- product id.

Remove Step From Look

URL: https://webserviceproj.herokuapp.com/api/removeStepFromLook/:look/:number
Result: JSON with message - success/error.
Method: GET
Params: look-name of look, number- step number.

Remove Step From Look

URL: https://webserviceproj.herokuapp.com/api/getLookSteps/:look
Result: JSON with message - success/error.
Method: GET
